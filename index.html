Juego Piedra Papel Tijera

Proyecto en Python - Autor: Yandry Alvarez

Descripción

Desarrollo del juego tradicional Piedra, Papel y Tijera en Python.

Este proyecto presenta el clásico juego de Piedra, Papel o Tijera, desarrollado en Python. Su propósito es mostrar de manera sencilla cómo aplicar la lógica condicional y el uso de listas, además de reforzar la práctica de programación básica mediante un ejemplo fácil de comprender.

Recientemente se implementó un sistema “Mejor de 3”, para ofrecer partidas más dinámicas y estratégicas. Esto requirió desarrollar un nuevo diagrama en Raptor llamado "PIEDRA PAPEL TIJERA MEJOR 3", que optimiza la lógica de control de rondas, manejo de puntuaciones y verificación de empates.
