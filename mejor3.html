Nuevas Novedades – Mejor de 3

La gran novedad del proyecto es la incorporación del sistema “Mejor de 3”, que transforma la experiencia de juego de manera significativa. Ahora, el jugador compite en rondas sucesivas, acumulando puntos hasta determinar un ganador definitivo.

Para soportar esta nueva lógica, se desarrolló un nuevo diagrama en Raptor llamado "PIEDRA PAPEL TIJERA MEJOR 3", que refleja el flujo actualizado:

  • Se añadieron variables de conteo de rondas y puntuaciones individuales.
  • Se implementó un bucle principal que controla las tres rondas y verifica el ganador de cada una.
  • Se ajustaron las condicionales para evaluar empates, victorias y derrotas dentro de cada ronda.

Código Python – Mejor de 3

import random

opciones = ["Piedra", "Papel", "Tijera"]

def jugar_mejor_de_3():
    puntaje_jugador = 0
    puntaje_computadora = 0

    for ronda in range(1, 4):
        print(f"\nRonda {ronda}")
        jugador = input("Elige Piedra, Papel o Tijera: ").title()
        computadora = random.choice(opciones)
        print("Jugador:", jugador)
        print("Computadora:", computadora)

        if jugador == computadora:
            print("Empate!")
        elif (jugador == "Piedra" and computadora == "Tijera") or \
             (jugador == "Papel" and computadora == "Piedra") or \
             (jugador == "Tijera" and computadora == "Papel"):
            print("¡Ganaste esta ronda!")
            puntaje_jugador += 1
        else:
            print("¡Perdiste esta ronda!")
            puntaje_computadora += 1

    print(f"\nResultados Finales: Jugador {puntaje_jugador} - Computadora {puntaje_computadora}")
    if puntaje_jugador > puntaje_computadora:
        print("¡Felicidades! Ganaste el juego Mejor de 3")
    elif puntaje_jugador < puntaje_computadora:
        print("¡La computadora ganó el juego Mejor de 3!")
    else:
        print("¡El juego terminó en empate!")

while True:
    print("\nBienvenido al juego PIEDRA PAPEL TIJERA MEJOR 3")
    jugar_mejor_de_3()
    if input("\n¿Deseas jugar otra vez? (si/no): ").title() != "Si":
        print("Gracias por jugar")
        break
